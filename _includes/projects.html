Manitoba Association of Foodbanks

A PHP-based web application designed to solve the issue of the foodbanks needing to monitor community statistics for their yearly report.

Previously, receiving food wasn't so easy-- you had to be crossed off of the foodbank's list, a list that required your Manitoba Health ID. The web app solved this issue by making an easy, fast, private and secure way for staff to search clients. A web application was chosen due to the ease at which you can access the contents, with a central server, the staff can login using their smartphone or any other internet device at any time. With other suggested solutions, they may have had the individual foodbanks buy expensive technologies to store data.

Project Evil

A Unity-based game written in C#, created, designed and written by me. It uses cellular automata to generate worlds completely randomly, but in a semi-realistic sort of way. While there's yet a long way to go, for this project, I've already begun to learn how to code within the Unity framework, animate, and design creatures/characters.

The game will feature a hardcore world where you must save it from the evil underneath, but be warned, if you die, that's it! Once you've died, you must start over from scratch. As you fight more monsters, you get more powerful items to further fight increasingly difficult monsters.
